# NVIM Robin

This is the configuration of [Neovim](https://github.com/neovim/neovim) for Robin.

## Requirements

Neovim >0.11
npm >18.0.0

## Installation

Make sure that neovim version is high enough.

> Important! Do not use the ubuntu packages. The neovim version is outdated.

You should build from source. You can use the following script to help:

```
curl -o- https://raw.githubusercontent.com/RobinDBL/nvim-robin/install-neovim.sh | bash
```
